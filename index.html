Core Features

Complete Privacy Stack

NyxLabs provides a comprehensive privacy solution that protects every aspect of your blockchain transactions.

Zero-Knowledge Proofs

Prove transaction validity without revealing sender, receiver, or amount. Built on Circom with Groth16/PLONK verification.

Circom SnarkJS Groth16

Stealth Addresses

One-time addresses for every transaction. Impossible to link multiple payments to the same recipient.

Relayer Network

Decentralized relayers submit transactions on your behalf, obscuring IP addresses and preventing metadata correlation.

Privacy SDK

TypeScript and Rust SDK with WebAssembly support for client-side proof generation and seamless integration.

Multi-Chain

Deploy on Ethereum, Polygon, Arbitrum, and Solana. Cross-chain privacy routing coming soon.

Optional Compliance

View keys and proof-of-funds generation for institutions. Privacy by default, disclosure by choice.
